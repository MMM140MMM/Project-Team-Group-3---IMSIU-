# Encryption Chat & Security Scanner 🛡️

A comprehensive security project that combines encrypted communication with network and link scanning tools. Developed as part of my cybersecurity studies to demonstrate practical implementation of cryptographic protocols and threat detection.

## 🚀 Project Overview
This project provides a secure environment for messaging using advanced encryption algorithms, alongside a built-in module for scanning links and network traffic to ensure they are free from potential threats.

## 🛠️ Technical Features
* **Strong Encryption:** Implements **RSA** algorithm for secure key exchange and channel authorization.
* **Link Scanner:** A specialized module to analyze URLs and verify their safety before interaction.
* **Graphical User Interface (GUI):** A user-friendly interface built using **Tkinter**.
* **Python Powered:** Developed entirely in Python, following security best practices.
